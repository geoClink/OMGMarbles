OMGMarbles
==========

A small, SpriteKit-based iOS game written in Swift. OMGMarbles is a lightweight arcade-style project that demonstrates core SpriteKit features such as physics-driven marbles, particle effects, scene management, and simple touch-driven gameplay. It includes art assets, a main game scene (`GameScene.sks` / `GameScene.swift`), and a minimal UIKit-based host (`GameViewController.swift`).

Features
- SpriteKit-based gameplay with physics and collisions
- Multiple marble sprites with assets included
- Particle effects and simple explosion animations
- Single-scene project designed for learning and quick prototyping

Quick Start (Build & Run)
1. Requirements
   - Xcode 12 or later
   - iOS SDK compatible with your Xcode version

2. Open the project
   - Double-click `OMGMarbles.xcodeproj` or open the folder in Xcode.

3. Build & Run
   - Select a simulator or a connected device and run (Cmd+R).

Project Structure
- `GameScene.sks` / `GameScene.swift` — main SpriteKit scene and game logic
- `GameViewController.swift` — view controller that presents the SpriteKit scene
- `Assets.xcassets` — included sprite and particle assets
- `AppDelegate.swift` — app lifecycle
